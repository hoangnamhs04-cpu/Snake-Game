# Snake Game
Xin chào thầy và tất cả các bạn! Nhóm 17 xin giới thiệu dự án cuối kỳ môn Lập trình Java – tựa game Rắn săn mồi (Snake Game) được nhóm thiết kế và phát triển hoàn toàn bằng Java. Với mong muốn tái hiện lại trò chơi cổ điển quen thuộc nhưng mang phong cách hiện đại hơn, nhóm đã tập trung vào việc xây dựng gameplay mượt mà, giao diện trực quan và hệ thống tính điểm hấp dẫn. Dự án không chỉ giúp nhóm vận dụng kiến thức về lập trình hướng đối tượng (OOP) mà còn rèn luyện kỹ năng làm việc nhóm, phân chia nhiệm vụ, xử lý lỗi và tối ưu chương trình trong quá trình phát triển. Đây là sản phẩm mà nhóm đã dành nhiều thời gian nghiên cứu, thử nghiệm và hoàn thiện. 🚀
